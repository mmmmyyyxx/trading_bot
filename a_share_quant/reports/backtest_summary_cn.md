# 回测绩效指标说明

| 指标字段 | 中文名称 | 当前数值 | 含义 |
| --- | --- | ---: | --- |
| `total_return` | 净总收益率 | -0.150163 | 扣除交易成本后，最终净值相对初始资金的累计收益率。 |
| `gross_total_return` | 毛总收益率 | -0.125406 | 将累计交易成本加回后的近似累计收益率，用于观察成本拖累。 |
| `annual_return` | 年化收益率 | -0.047696 | 把回测期净收益按 252 个交易日折算到一年的收益率。 |
| `annual_volatility` | 年化波动率 | 0.206052 | 每日净收益率标准差按 252 个交易日年化后的波动水平。 |
| `sharpe` | 夏普比率 | -0.133323 | 年化超额收益相对年化波动的比值；当前版本默认无风险利率为 0。 |
| `max_drawdown` | 最大回撤 | -0.325677 | 回测期间净值从历史高点到后续低点的最大跌幅。 |
| `calmar` | Calmar 比率 | -0.146453 | 年化收益率除以最大回撤绝对值，用于衡量收益和回撤的关系。 |
| `win_rate` | 胜率 | 0.491647 | 日收益率大于 0 的交易日占比。 |
| `turnover` | 累计换手率 | 29.786136 | 每日成交金额相对组合权益的换手率累计值。 |
| `average_turnover` | 平均日换手率 | 0.035502 | 回测期间每日换手率的平均值。 |
| `total_cost` | 累计交易成本 | 24757.033300 | 佣金、印花税、过户费和滑点影响的合计金额。 |
| `benchmark_return` | 基准收益率 | 0.271386 | 如果提供基准数据，则表示基准在同一时期的累计收益率。 |
| `excess_return` | 超额收益 | -0.421549 | 策略净收益率减去 benchmark 收益率。 |
| `tracking_error` | 跟踪误差 | 0.173928 | 策略日收益相对 benchmark 日收益差值的年化波动率。 |
| `information_ratio` | 信息比率 | -0.663495 | 年化相对收益除以跟踪误差，用于衡量相对收益稳定性。 |
| `beta` | Beta | 0.710191 | 策略收益相对 benchmark 收益的敏感度。 |
| `alpha` | Alpha | -0.100806 | 扣除 beta 暴露后的粗略年化超额收益估计。 |
| `up_capture` | 上涨捕获率 | 0.668416 | benchmark 上涨日中，策略平均收益相对 benchmark 平均收益的比例。 |
| `down_capture` | 下跌捕获率 | 0.764789 | benchmark 下跌日中，策略平均收益相对 benchmark 平均收益的比例。 |
| `relative_drawdown` | 相对最大回撤 | -0.449222 | 策略相对 benchmark 净值曲线的最大回撤。 |
| `monthly_win_rate_vs_benchmark` | 月度跑赢率 | 0.428571 | 策略月收益高于 benchmark 月收益的月份占比。 |
